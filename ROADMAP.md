# ROADMAP.md — Fourier 音频分析仪开发路线图

> 更新时间：2026-03-16

---

## 已完成功能（v1.x 现状）

### 核心分析引擎

- 自实现 Cooley-Tukey FFT（支持 256～16384 点）
- 多种窗函数：矩形 / 汉宁 / 汉明 / 布莱克曼 / 凯泽
- 频谱斜率补偿（-12～+12 dB/octave）
- A 加权滤波

### 可视化视图

- 实时频谱图（FFT 幅度谱）
  - 频率轴：线性 / 对数 / 十二平均律三种模式
  - 双指独立横/纵轴缩放，单指平移
  - 峰值检测与标记（前 10 峰）
  - 峰值保持线
- 波形示波器
  - 实时 PCM 波形滚动显示
  - 双指缩放时间轴 / 幅度轴
- 频谱瀑布图（时频图）
  - 环形缓冲区架构（60 秒 × 60fps = 3600 行）
  - 双指独立 X/Y 缩放，缩放中心为双指中点（2026-03-16 完成）
  - 彩虹 / 灰度 / 黑红 / 蓝绿四种配色
  - 历史回溯滚动（最多 60 秒）
- 声压级电平计（SPL Meter）

### 音频输入

- 麦克风实时录音（44100 Hz / 16-bit PCM）
- 音频文件分析（播放 + 实时 FFT）
- 系统音频捕获（MediaProjection，Android 10+）

### 实用功能

- PCM 录制保存
- 截图保存（长按）
- 增益控制滑块（0～200%）
- 实时信息面板（采样率 / FFT 大小 / 峰值频率 / 峰值幅度）
- 深色主题 + 默认横屏布局
- 本地离线构建（Gradle Cache + 本地 JDK/ADB）

## 近期计划（v1.x 迭代）

### 交互体验优化

- 瀑布图 Y 轴缩放惯性滑动（fling 支持）
- 频谱图双击放大聚焦至峰值频率
- 瀑布图 Y 轴时间标签随缩放自适应更新（当前已实现，可进一步优化对齐精度）
- 多点触控手势冲突优化（捏合 vs. 滑动切换页面）

### 功能完善

- FFT 大小运行时可调（目前固定或仅设置界面可改）
- 采样率可选（8000 / 16000 / 44100 / 48000 Hz）
- 频率范围裁剪（只显示用户关心的频段）
- 自定义频率标记（用户手动添加）

### 性能优化

- 瀑布图 `freqToXCache` 在 `addSpectrumData` 热路径的缓存命中率监控
- 频谱图大 FFT（≥4096）时移至后台线程预计算
- 降低 `onDraw` 中临时对象分配（减轻 GC 压力）

## 中期计划（v2.0）

### 分析功能增强

- 倍频程 / 1/3 倍频程频谱分析（RTA 模式）
- 频率响应测量（扫频 + 传递函数）
- THD（总谐波失真）检测
- 相位谱显示
- 频谱取平均（最大保持 / 指数平均 / 线性平均）
- 立体声双通道支持

### 数据导出

- 导出频谱为 CSV 数据文件
- 导出瀑布图为图片（带时间轴标注）
- 分享当前截图至系统分享菜单

### 高级可视化

- 频谱对比（参考曲线 vs. 实时曲线）
- 可调颜色映射编辑器（瀑布图）
- 3D 旋转瀑布图（实验性）
