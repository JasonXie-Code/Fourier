## 📝 实现说明

### 采样率可调实现
- 修改了`AudioRecorder`类，支持动态更改采样率

### 音频文件导入实现
- 创建了`AudioFileProcessor`类处理音频文件
- 使用`MediaExtractor`解码音频文件
- 支持多种音频格式（MP3、WAV、AAC等）
- 自动转换为单声道
- 文件模式和麦克风模式无缝切换

### 文件处理流程
1. 用户点击"导入音频文件"按钮
2. 打开文件选择器
3. 选择音频文件
4. `AudioFileProcessor`加载并解码文件
5. 显示文件信息
6. 开始逐帧读取音频数据
7. 输入到FFT处理流程
8. 实时显示频谱

## 💡 使用说明

### 采样率选择
1. 点击左上角"采样率"文本
2. 在弹出的对话框中选择采样率
3. 系统会自动重启音频处理

### 音频文件导入
1. 点击底部"导入音频文件"按钮
2. 选择音频文件（支持MP3、WAV等格式）
3. 文件加载后自动开始分析
4. 点击"麦克风模式"切换回实时麦克风输入

## ⚠️ 注意事项

1. **文件模式**: 在文件模式下无法录制音频
2. **采样率更改**: 更改采样率需要先停止录制
3. **文件格式**: 支持Android MediaExtractor支持的所有格式
4. **性能**: 大文件可能占用较多内存，建议使用较小的测试文件
