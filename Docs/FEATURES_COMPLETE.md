## 📱 使用方法

### FFT大小和重叠率
1. **选择FFT大小**: 点击"FFT大小"按钮
2. **调整重叠率**: 长按"FFT大小"按钮

### 采样率选择
- 点击左上角"采样率"文本

### 音频文件导入
1. 点击"导入音频文件"按钮
2. 选择音频文件（支持MP3、WAV等）
3. 文件自动开始分析
4. 点击"麦克风模式"切换回实时输入

### 播放录制文件
1. 点击"播放录制"按钮
2. 选择要播放的PCM文件
3. 文件自动开始播放并显示频谱
4. 播放完成后自动停止

### 频谱瀑布图
1. **切换到瀑布图**: 
   - 点击"频谱模式" → "示波器模式" → "瀑布图模式"
2. **返回频谱模式**: 
   - 长按"瀑布图模式"按钮
3. **清空瀑布图**: 
   - 切换模式时会自动清空

## 🎨 显示模式

### 频谱模式（默认）
- 实时显示FFT频谱
- 支持峰值检测和频率标记
- 线性和对数缩放

### 示波器模式
- 显示时域波形
- 实时音频波形可视化

### 瀑布图模式
- 时间-频率-幅度三维显示
- 热力图颜色映射
- 历史频谱滚动显示
- 最多显示200条历史数据（约4-5秒，取决于FFT大小和重叠率）

## 📊 性能说明

### FFT大小性能
- **512-4096**: 流畅（推荐日常使用）
- **8192**: 良好（高分辨率）
- **16384**: 可接受（专业分析）
- **32768**: 配合75%重叠率可达5.4 FPS

### 重叠率影响
- **0%**: 最低CPU占用，但更新频率最低
- **50%**: 默认推荐，最佳平衡
- **75%**: 最高刷新率，适合大FFT

## 📝 注意事项

1. **文件模式**: 文件模式下无法录制音频
2. **采样率更改**: 需要先停止录制
3. **瀑布图**: 切换模式时会清空历史数据
4. **播放文件**: 播放PCM文件时使用当前设置的采样率
5. **性能**: 大FFT和高重叠率会增加CPU占用
